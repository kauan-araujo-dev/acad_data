# 📊 AcadData

Sistema web para consulta e análise de dados de alunos de uma instituição de ensino, com foco em filtros dinâmicos e execução de consultas SQL.

🎯 Objetivo

Facilitar a exploração de dados acadêmicos, permitindo consultas rápidas e organizadas em uma base de alunos, sem necessidade de acessar diretamente o banco de dados.

⚙️ Funcionalidades

Listagem de alunos com paginação (10 em 10 registros)<br>
Filtros por ordem, nome, curso, semestre, período e data de matrícula<br>
Execução de consultas SQL personalizadas<br>
Carregamento progressivo de dados<br>
Visualização organizada das informações<br>
